# PRODIGY_DS_05

## Task 5: Traffic Accident Data Analysis

### Objective
Analyze traffic accident data to identify patterns related to casualty severity, age distribution, and accident-related trends using data visualization techniques.

### Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

### Steps Performed
1. Loaded and explored the accident dataset
2. Checked dataset structure and missing values
3. Performed statistical analysis
4. Visualized casualty severity distribution
5. Analyzed age distribution of casualties
6. Identified patterns in casualty types and demographics

### Key Insights
- Most accident casualties belonged to young and middle-aged groups.
- Casualty severity distribution showed moderate severity cases were more common.
- Accident patterns varied across gender and casualty types.
- Data visualization helped in identifying important accident trends.

### Conclusion
This project demonstrated how traffic accident datasets can be analyzed using Python and visualization techniques to identify meaningful patterns and trends.
